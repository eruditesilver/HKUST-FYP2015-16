Knowledge Point Graph

To implement a module that allows instructors to create a knowledge point graph (containing knowledge points, also known as, concepts) for a particular course and also allows them to link each knowledge point to the associated resources (lecture notes, videos, questions and self-tests)

Q&A Messenger

To implement a module called the “Q&A message box” for posting questions and handling replies. Students can ask questions about the course content. Instructors can choose to give text replies immediately or later.The Q&A message box can be used both in-class and after-class

Self Test

To create a module for instructors to create self-tests for knowledge points Students can use the self-tests for self-evaluation. Instructors can use the self-tests for evaluating students’ progress and finding out common misconceptions/problems so as to improve the course
